# Frontend (Next.js + React + pnpm)

Этот проект использует **Next.js**, **React**, **pnpm** и **Node.js**. Ниже приведена краткая инструкция по установке окружения и запуску приложения.

---

## Требования

Перед началом убедитесь, что установлены:

### 1. Node.js  
Рекомендуем установить последнюю LTS-версию:  
https://nodejs.org/

Проверка версии:
```bash
node -v
````
```bash
npm -v
````

### 2. Pnpm

Установка через npm:

```bash
npm install -g pnpm
```

Проверка версии:

```bash
pnpm -v
```

---

## Установка и запуск проекта

> Из папки проекта

### 1. Установка зависимостей

```bash
pnpm install
```

### 2. Режим разработки

```bash
pnpm dev
```

Приложение будет доступно по адресу:
[http://localhost:3000/](http://localhost:3000/)
